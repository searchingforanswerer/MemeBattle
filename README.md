# 🤗 表情包大作战 - 随机题目生成器

## 📋 项目简介

表情包大作战，就是每个玩家有一些Meme手牌，给定一个场景牌，玩家需要打出对应的Meme。由于大家都把自己喜欢的表情包存在微信和QQ中，所以本项目提供了一个场景的随机题目生成器，可以在本网站抽取题目，然后在微信中和朋友使用符合场景的表情包。
本项目是一个基于 HTML/CSS/JavaScript 的表情包大作战随机题目生成器，支持基础题库和用户自定义题目的管理功能。

## ✨ 主要功能

- 🎲 随机题目抽取
- ➕ 自定义题目添加
- 📋 自定义题目管理（查看、编辑、删除）
- 🗑️ 批量操作功能
- 🎨 响应式设计
- 📱 本地数据持久化

## 🚀 使用方法

### 基础功能
1. 点击"🎲 获取随机题目"获取随机题目
2. 在输入框中输入新题目并点击"添加题目"

### 自定义题目管理
1. 点击"📋 查看自定义题目"打开管理弹窗
2. 支持单个编辑/删除
3. 支持批量选择和操作

## 📝 自定义配置

### 修改题库
编辑 `questions.json` 文件，按照以下格式添加题目：

```json
{
  "questions": [
    "题目1",
    "题目2",
    "题目3"
  ]
}
```
